#  Zomato Restaurant Analysis Dashboard (Power BI)

##  Project Overview
This project focuses on analyzing restaurant data from Zomato to understand customer preferences, restaurant performance, and market trends.

The dashboard provides interactive visualizations to explore restaurant ratings, cost, cuisines, and services like online delivery and table booking.

---

##  Problem Statement
To analyze restaurant data and identify key factors affecting ratings, cost, and customer engagement across different countries and cities.

---

##  Key Metrics
- Total Restaurants: 9551
- Average Rating: 3.01 
- Average Cost for 2: ₹1199

---

##  Dashboard Features

###  1. Interactive Filters
- Country selection  
- City selection (Nagpur selected in view)  
- Rating color filter  
- Online delivery availability  
- Table booking availability  

 Helps users dynamically explore data based on preferences.

---

###  2. Treemap Visualization
- Displays restaurant distribution by name  
- Color-coded based on rating performance  
  -  Green → Good ratings  
  -  Yellow → Average  
  -  Red → Low ratings  

 Helps quickly identify top and low-performing restaurants.

---

###  3. KPI Cards
- Total restaurant count  
- Average rating  
- Average cost for 2  

Provides quick business insights at a glance.

---

###  4. Geo Map Visualization 
- Shows restaurant distribution across continents  
- Helps understand global presence of restaurants  

---

###  5. Cost & Rating Analysis
- Gauge charts used to display:
  - Average cost  
  - Average rating  

 Makes it easy to interpret performance visually.

---

###  6. Cuisine Analysis
- Lists different cuisines like:
  - Chinese  
  - Continental  
  - European  
  - Fast Food  

 Helps understand popular food trends.

---

##  Tools & Technologies Used

###  Power BI
- Developed interactive and dynamic dashboards  
- Created visualizations like Treemap, Map, KPI Cards, and Gauge charts  
- Applied slicers for filtering (Country, City, Services, Ratings)  
- Designed clean and user-friendly dashboard UI  

---

###  DAX (Data Analysis Expressions)
- Used for creating calculated measures and KPIs  
- Implemented formulas for:
  - Average Rating  
  - Average Cost for Two  
  - Total Restaurants count  
- Enabled dynamic calculations based on filters and user interaction  

---

###  Power Query Editor
- Performed data cleaning and transformation  
- Handled missing/null values  
- Renamed and formatted columns  
- Merged and structured datasets for analysis  
- Prepared raw data for efficient reporting  

---

###  Microsoft Excel
- Used as a data source  
- Organized raw datasets into structured format  
- Assisted in initial data preprocessing  

---

###  Data Visualization Techniques
- Applied color coding for rating insights  
- Used KPI indicators for quick understanding  
- Implemented interactive filtering for better user experience
